# 머신러닝 퀴즈

### 질문 1: 한 연구팀이 자연어 처리(NLP)를 사용하여 논문을 주제별로 그룹화하려고 합니다. 어떤 서비스를 사용해야 할까요?

1.  번역하다
2.  이해하다
3.  법률
4.  인식

>> 정답: 2번
>>
>> 메모: Amazon Comprehend 이게 NLP 서비스인데, 토픽 모델링이란 기능으로 논문 같은 문서들을 읽고 '아, 이건 이 주제네', '저건 저 주제네' 하면서 알아서 주제별로 그룹을 묶어주는 거임.

---

### 질문 2: 개발자가 머신 러닝 모델을 빠르게 구축, 학습, 배포하고 싶어합니다. 어떤 서비스를 사용할 수 있을까요?

1.  세이지메이커
2.  폴리
3.  이해하다
4.  개인화

>> 정답: 1번 (세이지메이커)
>>
>> 메모: 세이지메이커는 ML 모델 만들고, 학습시키고, 배포하는 거까지 한 방에 다 해주는 플랫폼.

---

### 질문 3: Amazon Polly를 사용하면 텍스트를 음성으로 변환할 수 있습니다. 두 가지 중요한 기능이 있습니다. 첫 번째는 단어의 발음을 사용자 지정할 수 있는 ………………..입니다(예: "Amazon EC2"를 "Amazon Elastic Compute Cloud"로 변경). 두 번째는 ………………..입니다. 숨소리, 속삭임 등 단어를 강조할 수 있습니다.

1.  음성 합성 마크업 언어(SSML), 발음 사전
2.  발음 사전, 보안 어설션 마크업 언어(SAML)
3.  발음 사전, 음성 합성 마크업 언어(SSML)
4.  보안 어설션 마크업 언어(SAML), 발음 사전

>> 정답: 3번
>>
>> 메모: 첫 번째 빈칸은 발음 사전(Lexicons). 두 번째는 SSML(음성 합성 마크업 언어), 이걸로 '여기선 좀 쉬고', '여긴 속삭이듯이' 이렇게 디테일하게 조절해서 사람처럼 말하게 만들기 가능.

---

### 질문 4: 이미지와 비디오에서 사물, 사람, 텍스트 또는 장면을 찾고 싶습니다. 어떤 AWS 서비스를 사용해야 할까요?

1.  인식
2.  폴리
3.  켄드라
4.  법률

>> 정답: 1번 (인식)
>>
>> 메모: Amazon Rekognition(인식)은 사진이나 영상 던져주면 '여기 사람 있네', '이건 고양이', '글자는 이거 적혀있음' 하고 다 찾아주는 거.

---

### 질문 5: 한 의료 회사가 의사 소견서, 임상시험 보고서... 비정형 의료 텍스트에서 정보를 탐지, 추출 및 분석하는 솔루션을... HIPAA... 보호 건강 정보(PHI)를 식별... 어떤 AWS 서비스를 사용해야 할까요?

1.  Amazon Comprehend Medical
2.  아마존 레코그니션
3.  아마존 폴리
4.  아마존 번역

>> 정답: 1번
>>
>> 메모: Amazon Comprehend Medical. 이름 그대로 '의료용' NLP임.

---

### 질문 6: ...Amazon Connect와 Amazon Lex를 사용하여 전화를 받고... 모든 통화는 검토를 위해 녹음되어야 합니다. 하지만 통화를 저장하기 전에 개인 식별 정보(PII)를 삭제해야 합니다. ...어떤 솔루션을 추천해 주시겠습니까?

1.  아마존 폴리
2.  아마존 트랜스크라이브
3.  아마존 인식
4.  아마존 번역

>> 정답: 2번 (아마존 트랜스크라이브)
>>
>> 메모: Amazon Transcribe는 전화 녹음 같은 음성을 텍스트로 바꿔주는 서비스임. 근데 부가 기능으로 PII Redaction이 있음. 텍스트로 바꾸면서 '어, 이거 주민번호네?', '이건 카드번호네?' 하는 것들을 찾아서 `[PII]` 이렇게 가려주는(마스킹) 거.

---

### 질문 7: ...이미지 및 비디오 공유 플랫폼... 부적절하고 불쾌한 이미지와 비디오를 자동으로 감지하고, 플래그가 지정될 항목에 대한 최소 신뢰 임계값을 설정... 어떤 AWS 서비스가 이러한 요구 사항을 충족할 수 있을까요?

1.  아마존 폴리
2.  아마존 번역
3.  아마존 렉스
4.  아마존 레코그니션

>> 정답: 4번 (아마존 레코그니션)
>>
>> 메모: Amazon Rekognition 이건 '콘텐츠 중재(Moderation)' 기능임. 사진이나 영상 보고 '오 이거 좀 야한데?', '이거 폭력적인데?' 하고 잡아내는 거.

---

### 질문 8: 다음 서비스 중 머신 러닝을 활용한 문서 검색 서비스는 무엇입니까?

1.  번역하다
2.  켄드라
3.  이해하다
4.  폴리

>> 정답: 2번 (켄드라)
>>
>> 메모:  ML 기반 검색 엔진임. 그냥 키워드 검색이 아니라 '작년 4분기 매출 보고서 어디있어?' 하고 자연어로 물어봐도 문서를 찾아주는 서비스.

---

### 질문 9: 어떤 AWS 서비스를 사용하면 음성을 텍스트로 쉽게 변환할 수 있나요?

1.  연결하다
2.  번역하다
3.  고쳐 쓰다
4.  폴리

>> 정답: 3번 (고쳐 쓰다)
>>
>> 메모: '고쳐 쓰다'가 번역이 좀 이상한데, Amazon Transcribe(트랜스크라이브)를 말하는 거 같음. 음성을 텍스트로 바꿔주는 거.

---

### 질문 10: Amazon Transcribe를 사용하면 딥 러닝을 사용하여 텍스트를 실제 음성으로 바꿀 수 있습니다.

1.  진실
2.  거짓

>> 정답: 2번 (거짓)
>>
>> 메모: Transcribe는 음성을 텍스트로 바꿔주는 거임. 반대는 Amazon Polly.

### 질문 11: 어떤 회사에서 문서를 자연스럽고 정확한 표현으로 여러 언어로 변환하고 싶어 합니다. 어떤 언어를 사용해야 할까요?

1.  고쳐 쓰다
2.  폴리
3.  번역하다
4.  단어 번역기

>> 정답: 3번 (번역하다)
>>
>> 메모: Amazon Translate. 신경망 기계 번역(NMT) 써서 여러 나라 말로 자연스럽게 번역해주는 서비스.

### 질문 12: 한 스타트업이 맞춤형 사용자 경험을 빠르게 구축하고 싶어합니다. 어떤 AWS 서비스가 도움이 될까요?

1.  개인화
2.  켄드라
3.  연결하다

>> 정답: 1번 (개인화)
>>
>> 메모: Amazon Personalize가 맞춤형 사용자 경험 즉, 개인화 추천을 만들어주는 서비스.

---

### 질문 13: 한 회사가 음성을 텍스트로 변환하고 고객의 의도를 파악하는 챗봇을 구현하려고 합니다. 어떤 서비스를 사용해야 할까요?

1.  고쳐 쓰다
2.  인식
3.  연결하다
4.  법률

>> 정답: 4번 (법률)
>>
>> 메모: Amazon Lex**임. 이게 바로 챗봇 만드는 서비스임. Lex 안에는 음성을 텍스트로 바꾸는 기능이랑, '예약하고 싶어' 같은 말의 뜻을 파악하는 기능(NLU)이 다 들어있음.
